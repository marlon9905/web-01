Ejemplo de Listas HTML

En HTML, las listas permiten organizar información de manera estructurada. Hay dos tipos principales: listas desordenadas (con viñetas) y listas ordenadas (con números o letras).

Lista Desordenada (<ul>)

Una lista desordenada comienza con la etiqueta <ul> (unordered list). Cada elemento de la lista se define con <li> (list item). Por defecto, los elementos se marcan con viñetas circulares.

  • Elemento 2
  • Elemento 3

Lista Desordenada con Diferentes Viñetas

El atributo type en <ul> permite cambiar el estilo de las viñetas: disc (predeterminado), circle (círculo hueco) o square (cuadrado). Nota: Este atributo está obsoleto en HTML5; se recomienda usar CSS para estilos personalizados.

  • Elemento con viñeta cuadrada
  • Otro elemento

Lista Ordenada (<ol>)

Una lista ordenada comienza con la etiqueta <ol> (ordered list). Cada elemento se define con <li>. Por defecto, los elementos se numeran con números (1, 2, 3...).

  1. Coffee
  2. Tea
  3. Milk

Lista Ordenada con Diferente Numeración

El atributo type en <ol> permite cambiar el tipo de numeración: 1 (números, predeterminado), A (letras mayúsculas), a (letras minúsculas), I (números romanos mayúsculos), o i (números romanos minúsculos). También está obsoleto en HTML5; usa CSS en su lugar.

Lista Anidada

Las listas pueden anidarse colocando una lista dentro de un elemento <li>. Esto es útil para crear subniveles de información, como categorías y subcategorías.

  • Frutas
    • Manzana
    • Naranja
    • Banana
  • Bebidas
    1. Agua
    2. Jugo
