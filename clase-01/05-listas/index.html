Listas en HTML

Aprende los 3 tipos de listas básicas

1. Lista Sin Orden (ul)

Se usa cuando el orden NO importa

Código:

  • Manzana
  • Banana
  • Naranja

Se ve así:

  • Manzana
  • Banana
  • Naranja

2. Lista Con Orden (ol)

Se usa cuando el orden SÍ importa

Código:

  1. Despertar
  2. Bañarse
  3. Desayunar

Se ve así:

  1. Despertar
  2. Bañarse
  3. Desayunar

3. Listas Dentro de Listas

Puedes poner una lista dentro de otra lista

Código:

  • Deportes
    • Fútbol
    • Baloncesto
  • Música
    • Rock
    • Pop

Se ve así:

  • Deportes
    • Fútbol
    • Baloncesto
  • Música
    • Rock
    • Pop

Cambiar el estilo con CSS

Cambiar las bolitas de ul:

ul { /* ul es el selector para listas sin orden */ list-style-type: square; /* Cambia las bolitas por cuadrados */ }
  • Ahora son cuadrados
  • En vez de círculos

Cambiar los números de ol:

ol { /* ol es el selector para listas con orden */ list-style-type: lower-alpha; /* Cambia números por letras */ }
  1. Ahora son letras
  2. En vez de números

Para recordar:

  • ul = lista SIN orden (con bolitas)
  • ol = lista CON orden (con números)
  • li = cada elemento de la lista
  • Puedes poner listas dentro de otras listas
